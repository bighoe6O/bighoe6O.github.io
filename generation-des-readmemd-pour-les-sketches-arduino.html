Génération des README.md pour les sketches arduino

Published: Mon 08 January 2018
By Moi

In Archive.

tags: Arduino 

for dir in *; do if [ -d "$dir" ]; then if [ ! -f "$dir/README.md" -a -f "$dir/$dir.ino" ]; then echo $dir;  cat README.tmpl | sed -e "s#@title@#$dir#" > $dir/README.md; fi; fi; done

README.tmpl

---
title: @title@
layout: inopage
---
